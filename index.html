Introduction

HTML is the standard markup language for documents designed to be displayed in a web browser.

It can be assisted with technologies such as CSS and Javascript.

HTML elements are the building blocks of HTML pages. The latest version of HTML is HTML5 which was released in October 2014.

Structure

HTML pages are text documents which uses tags to give the information they surround special meaning. These tags are referedded to as elements

Opening tags can carry attributes, which tell us more about the content of that element. Attributes require a name and a value. To learn, HTML you need to know what tags are available to use, what they do, and where they can go.

Text

There are various types of tags that compose a webpage. These tags are also known as markup, they provide meaning and structure to the page.

  • Structural Markup The elements that you can use to describe both headings and paragraphs.
  • Semantic Markup It provides extra information such as where emphasis is placed in a sentence, something is a quote...

Structural Markup

Headings H1 is a Heading tag where there six "levels" of heading from 1 to 6 with 6 being the smallest. Paragraphs p tag is used to create a paragraph. Bold & Italic b is used the make characters appear bold. i is used to make characters appeaer italic.

Semantic Markup

Strong & Emphasis strong is used to emphasis content that has strong importance. em indicates some emphasis on an element which slightly changes the meaning in a sentence.
Lists
Ordered Lists ol tag is used to create an ordered list. Unordered Lists ul tag is used to create an unordered list.
Links
"a href="http://www.google.com" Click Here to go Google.

Links are the defining feature of the internet as they allow you to move from one web page to another or even other parts of a website.

Links are created using the a element. Users can click on anything in the a tag.

href= " " is used to specify which page we want the link to lead to.

With links we're able to do a couple of things

  • Link to other sites
  • Link to other pages on the same website
  • Link to a specific element on the same page
  • Link to a specific part of a different page
  • Open a link in a new window.
